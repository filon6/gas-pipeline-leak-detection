# Определение координаты утечки газа в трубопроводе  
(физическая модель + машинное обучение)

## Описание проекта

В данном проекте исследуется задача **локализации утечки газа в трубопроводе**.

Подход основан на сочетании:

- физической модели течения газа
- генерации синтетического датасета
- модели машинного обучения (нейросети)

Цель — определить **координату утечки вдоль трубопровода** на основе измеряемых параметров:

- давления на входе и выходе
- расхода газа на входе и выходе
- геометрических параметров трубы
- физических характеристик газа


---

# Физическая модель

В основе лежит модель течения газа в трубопроводе.

Используются следующие формулы:

### Скорость потока

v = 4Q / (π d²)

где  
Q — расход газа  
d — диаметр трубы

---

### Число Рейнольдса

Re = (d · v · ρ) / η

где  

ρ — плотность газа  
η — динамическая вязкость

---

### Коэффициент трения

для ламинарного режима:

λ = 64 / Re

для турбулентного режима:

λ = 0.11 ((k_e / d) + (68 / Re))^0.25

---

### Коэффициент сопротивления трубопровода

k = 626.1 ρ λ / d⁵

---

### Формула координаты утечки

x = ((P_start - P_end)/k - L Q_end²) / (Q_start² - Q_end²)

где  

P_start — давление на входе  
P_end — давление на выходе  
Q_start — расход на входе  
Q_end — расход на выходе  
L — длина трубы

---

# Генерация датасета

Реальные данные утечек получить сложно, поэтому используется **синтетический датасет**.

Алгоритм генерации:

1. случайно выбираются параметры трубы  
2. задаётся истинная координата утечки  
3. вычисляется давление на выходе из физической модели  
4. добавляется шум измерительных датчиков  
5. проверяются физические ограничения

В датасет входят:

- параметры трубы
- измеренные давления
- измеренные расходы
- дополнительные признаки
- координата утечки

Размер датасета по умолчанию:

12000 наблюдений.

---

# Машинное обучение

Используется **полносвязная нейросеть (MLP)**.

Архитектура:

Input → 64 → 64 → 32 → 1

Активации:

ReLU

Выход:

Sigmoid (предсказывается нормированная координата x/L)

---

# Обучение модели

Функция потерь:

MSE (Mean Squared Error)

Оптимизатор:

Adam

Обучение:

200 эпох

---

# Метрики качества

Используются стандартные метрики регрессии:

MAE — средняя абсолютная ошибка  
RMSE — корень из среднеквадратичной ошибки  
R² — коэффициент детерминации

Метрики вычисляются:

1. для нормированной координаты (x/L)
2. для координаты в метрах

Также проводится сравнение с **аналитической формулой**.
